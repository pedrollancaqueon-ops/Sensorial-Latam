# Diseño y UX

## Principios
- Mobile-first: se usa desde el celular con cámara
- Evitar look "template" genérico
- Transiciones suaves al enviar
- Tipografía limpia y legible

## Paleta de colores LATAM
- Rojo principal: `#E31837`
- Azul oscuro: `#00175A`
- Blanco: `#FFFFFF`
- Gris neutro (fondo/bordes): a definir

## Componentes clave
- **Pantalla de cámara:** botón grande de captura, preview del encuadre
- **Loader "Analizando plato...":** animación breve (1–3 seg), no spinner aburrido
- **Card de confirmación:** nombre del plato sugerido, botón "Confirmar" + opción de editar
- **Formulario sensorial:** sliders o botones 1–5 con descriptores visuales (no solo número)
- **Confirmación de envío:** animación o mensaje visual claro (no alert de JS)
